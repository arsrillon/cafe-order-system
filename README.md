# Cafe Order System

A simple Java console application for managing cafe orders.  
Created as a pet project to practice Object-Oriented Programming (OOP).

## Features
- Add food or drinks to an order  
- Remove items from an order  
- Search orders  
- Exit the program  

## Technologies
- Java 21  
- Console-based interface (no external libraries)
